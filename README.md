# 使用技術及插件
* UI框架
    * [Bootstrap](https://getbootstrap.com/)
* 前端框架
    * [Vue](https://vuejs.org/)
* 首頁標題打字效果
    * [typed-js](https://github.com/mattboldt/typed.js/)
* 輪播效果
    * [swiper](https://swiperjs.com/)
* Animation
    * [aos](https://michalsnik.github.io/aos/)
